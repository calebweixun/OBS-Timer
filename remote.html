OBS 操作說明

加入 Playclock 到 OBS

1
在場景清單中右鍵 → 加入來源 → 瀏覽器
2
將 URL 貼上：
http://localhost:8080/playclock.html
3
寬度 1280，高度 720
4
勾選 「場景啟動時重整瀏覽器」
5
按確定，Playclock 即連上 Remote

用其他裝置遙控

在手機或平板的瀏覽器輸入：

http://你的IP:8080/remote.html

查詢本機 IP：macOS 系統設定 → 網路；Windows ipconfig

Preset Pad 操作

▶
點擊 Pad → 立即啟動倒數，並套用該 Pad 的樣式、閃爍、邊框設定
✎
Hover Pad → 點右上角鉛筆圖示 → 編輯完整設定
＋
點最後一格加號 → 新增 Preset

樣式設定說明

每個 Preset 各自儲存：

畫面標題 文字色 / 底色 字體 結束閃爍顏色 邊框樣式 邊框閃爍

更換 Port

啟動時帶入 port 參數：

./start.sh 9191

或設定環境變數 PORT=9191 node server.js
